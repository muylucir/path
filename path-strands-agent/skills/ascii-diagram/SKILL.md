# ASCII Diagram (한글/영어 혼합)

한글과 영어가 섞인 ASCII 다이어그램을 정확히 정렬하여 생성합니다.

## 핵심 규칙

### 0. 오른쪽 테두리 금지 (필수!)
**한글/영어 혼합 시 오른쪽 세로 테두리(│)가 어긋남.** 폰트마다 폭이 달라 정렬 불가능.

❌ **금지 (닫힌 박스)**:
```
┌──────────────┐
│  API 호출    │  ← 오른쪽 │가 어긋남
└──────────────┘
```

✅ **권장 (열린 박스)**:
```
┌──────────────
│  PRIMARY PATTERNS
├──────────────
│  ReAct + Tool Use
│  (추론)   (도구)
└──────────────
```

✅ **권장 (가로선만)**:
```
═══════════════════════════
  PRIMARY PATTERNS
═══════════════════════════
  ReAct        Tool Use
  (추론)       (도구)
───────────────────────────
```

✅ **권장 (마크다운 테이블)**:
```
| 패턴 | 설명 |
|------|------|
| ReAct | 추론 기반 |
| Tool Use | 도구 호출 |
```

### 1. 권장 최대 폭
```
권장 다이어그램 폭: 60자 이내
```

### 2. 출력 형식
**반드시** 코드 블록으로 감싸기:
````markdown
```
내용
```
````

## Quick Reference

### 박스 (열린 스타일)
```
┌─────────────────────
│  PRIMARY PATTERNS
├─────────────────────
│  User Input
│  데이터 검증
│  API 호출
└─────────────────────
```

### 플로우차트 (화살표 중심)
```
    [ 시작 ]
        │
        ▼
    < 조건? >
     │      │
    Yes     No
     │      │
     ▼      ▼
  [처리A]  [처리B]
     │      │
     └──┬───┘
        ▼
    [ 종료 ]
```

### 시퀀스 다이어그램 (세로선 중심)
```
  Client      Server        DB
    │           │           │
    │──요청────▶│           │
    │           │──쿼리────▶│
    │           │◀──결과────│
    │◀──응답────│           │
    │           │           │
```

### 테이블 (마크다운 권장)
```
| 항목      | 값   | 상태   |
|-----------|------|--------|
| 데이터    | 100  | 완료   |
| API 연동  | 50   | 진행중 |
| 테스트    | 25   | 대기   |
```

### 트리 구조 (그대로 사용 가능)
```
프로젝트/
├── src/
│   ├── components/
│   └── pages/
├── public/
└── package.json
```
